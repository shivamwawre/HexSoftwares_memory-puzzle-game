# 🧠 Memory Puzzle Game

A fun card matching game built using Python and Tkinter.

## 🎮 Features
- Full screen UI
- Animated background
- Card flip animation
- Timer countdown
- Restart option
- Win & Game Over detection

## 🚀 How to Run
1. Install Python
2. Run:
   python memory_game.py

## 🛠 Tech Used
- Python
- Tkinter
